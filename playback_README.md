# Playback Speed

A Python program that slows down text by replacing every space with `...`, mimicking a dramatic pause effect.

## What it does

Takes a sentence as input and returns it with all spaces replaced by `...`, giving the effect of slow, deliberate playback.

## How to run

```bash
python playback.py
```

**Example:**
```
This is fun
This...is...fun
```

## Concepts used

- User input with `input()`
- String `.replace()` method
